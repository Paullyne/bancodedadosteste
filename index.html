Aula 01: Hospedagem Web e Sistemas de Gerenciamento de Conteúdo (CMS)

1. Hospedagem de Sites na Internet

A hospedagem web é o serviço essencial que permite que indivíduos e organizações disponibilizem sites ou páginas na Internet. Provedores de hospedagem fornecem as tecnologias e infraestrutura necessárias para que o conteúdo seja acessível globalmente.

Modelos Tradicionais de Hospedagem

  • Hospedagem Compartilhada: Múltiplos sites compartilham o mesmo servidor físico, tornando-a econômica mas suscetível a variações de desempenho.
  • VPS (Virtual Private Server): Servidor físico particionado em máquinas virtuais isoladas, oferecendo controle e recursos dedicados.
  • Servidor Dedicado: Aluguel exclusivo de um servidor físico inteiro, ideal para alta performance e controle total.
Infraestrutura de Suporte: O DNS (Domain Name System) traduz domínios em IPs; FTP transfere arquivos; HTTPS garante segurança na comunicação.

Cloud Computing

Na nuvem, servidores virtuais interconectados formam uma infraestrutura elástica, diferentemente da hospedagem tradicional fixa.

Vantagens Principais:
• Escalabilidade: Ajuste recursos instantaneamente conforme demanda.
• Pagamento por Uso: Cobrança baseada em consumo real (CPU, RAM, tráfego).
• Provedores: AWS, Google Cloud, Microsoft Azure, DigitalOcean.

Escalabilidade Vertical vs Horizontal

  • Vertical: Aumentar potência (RAM/CPU) do servidor atual.
  • Horizontal: Adicionar mais servidores à rede.

2. Sistemas de Gerenciamento de Conteúdo (CMS)

CMS é um software que permite criar, editar e gerenciar conteúdo web sem programação avançada, separando Frontend (interface visual) do Backend (processamento e dados).

Tipos de CMS

  • Open Source: Código aberto e gratuito - WordPress, Joomla, Drupal.
  • SaaS: Plataformas hospedadas e pagas - Wix, Shopify.
  • Headless CMS: Backend via API, sem frontend acoplado.
Componentes Chave:
• Templates: Layouts visuais pré-definidos.
• Plugins/Extensões: Funcionalidades adicionais.
• Customização: Edição via CSS (estilo) ou painel admin.
• Banco MySQL: Armazenamento de dados (ex: wp-config.php no WordPress).

Instalação Típica

  • Criação de banco MySQL.
  • Upload via FTP (FileZilla).
  • Configuração inicial e escolha de template.
  • Testes locais com XAMPP/WAMP.
